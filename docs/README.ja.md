# 🍅 PomodoroNotch

**作業の邪魔をしないPomodoroタイマー。** メニューバーに常駐、キーボード操作、5MB未満のRAM使用。アカウント不要、クラウド不要。

## なぜまたPomodoroアプリ？

多くのPomodoroタイマーは、500MBのRAMを消費するElectronアプリか、ブラウザのタブに埋もれるWebツールです。優れたタイマーは、必要になるまで見えず、必要なときに瞬時に操作できるべきです。

PomodoroNotchは違います：ネイティブSwiftUIメニューバーアプリ、`⌃⌥⌘P`グローバルショートカット、ネットワーク通信なし、7ファイル・約1,000行のソースコード。

## メニューバーの状態

| 状態 | 表示 | 色 |
|------|------|------|
| 待機中 | `·` | 灰色・半透明 |
| 集中 | `24:59` または `🍅 25m` | オレンジ #FF6B35 |
| 短い休憩 | `☕ 5m` | 緑 #34C759 |
| 長い休憩 | `🛋 15m` | 緑 #34C759 |
| 一時停止 | 点滅アニメーション | 灰色 #8E8E93 |

## インストール

[Releases](https://github.com/raymondjxj/PomodoroNotch/releases/latest) からDMGをダウンロード。
```bash
git clone https://github.com/raymondjxj/PomodoroNotch.git && cd PomodoroNotch && make run
```

## グローバルショートカット

| キー | 操作 |
|------|------|
| `⌃⌥⌘P` | 開始 / 一時停止 / 再開 |
| `⌃⌥⌘S` | 現在のフェーズをスキップ |
| `⌃⌥⌘R` | リセット |

システム設定 → プライバシーとセキュリティ → アクセシビリティで許可が必要です。

## 機能

- **ドロップダウンパネル**: 32ptの大きなカウントダウン、コントロール、今日の統計、週間グラフ
- **12種類のチック音**: 設定でライブプレビュー可能
- **通知**: サウンド+バナー / サウンドのみ / バナーのみ / オフ
- **統計**: ローカルJSONファイルに保存
- **状態の永続化**: フェーズ変更ごとにUserDefaultsに保存、再起動後も正確に復元
- **11言語対応**: システム言語の自動検出または手動選択

## アーキテクチャ

7つのSwiftファイル、ゼロ外部依存。再帰的enum `TimerPhase` による状態機械、`NotificationCenter` によるモジュール分離、`@AppStorage` による設定管理。

## FAQ

**メニューバーに表示されない？** macOS 26+ではシステム設定 → コントロールセンター → PomodoroNotchを表示に設定。

**ショートカットが効かない？** アクセシビリティ許可を確認してください。
