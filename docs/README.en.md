# PomodoroNotch
> A minimal Pomodoro timer for the macOS menu bar.

## Features
- Color-coded menu bar countdown (orange=focus, green=break, gray=paused)
- Two display modes: `24:59` or `🍅 25m`
- 12 tick sounds with live preview
- Global shortcuts `⌃⌥⌘P/S/R`
- System notifications, sleep-aware, launch at login
- 11 languages
