# PomodoroNotch
> Минималистичный Pomodoro-таймер для строки меню macOS.

## Возможности
- Цветной обратный отсчёт (оранжевый=фокус, зелёный=перерыв, серый=пауза)
- `24:59` или `🍅 25m` | Фокус→Короткий перерыв→Фокус→Длинный перерыв→цикл
- 12 звуков тиканья | Горячие клавиши `⌃⌥⌘P/S/R`
- Системные уведомления | 11 языков
