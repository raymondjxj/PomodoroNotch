# 🍅 PomodoroNotch

**Pomodoro-таймер, который не мешает.** В строке меню. Управление с клавиатуры. Менее 5 МБ ОЗУ. Без учётных записей, без облака.

## Почему ещё одно Pomodoro-приложение?

Большинство Pomodoro-таймеров — либо раздутые Electron-приложения (500 МБ ОЗУ, подписки), либо веб-инструменты, теряющиеся во вкладках браузера. Хороший таймер должен быть невидимым, пока не понадобится, и мгновенным, когда нужен.

PomodoroNotch другой: нативное приложение для строки меню на SwiftUI, глобальный шорткат `⌃⌥⌘P`, ноль сетевых запросов, 7 файлов исходного кода (~1 000 строк).

## Состояния строки меню

| Фаза | Отображение | Цвет |
|------|-------------|------|
| Ожидание | `·` | Серый, прозрачный |
| Фокус | `24:59` или `🍅 25m` | Оранжевый #FF6B35 |
| Короткий перерыв | `☕ 5m` | Зелёный #34C759 |
| Длинный перерыв | `🛋 15m` | Зелёный #34C759 |
| Пауза | Пульсирующая анимация | Серый #8E8E93 |

## Установка

Скачайте DMG со страницы [Releases](https://github.com/raymondjxj/PomodoroNotch/releases/latest).

## Глобальные горячие клавиши

| Клавиши | Действие |
|---------|----------|
| `⌃⌥⌘P` | Старт / Пауза / Продолжить |
| `⌃⌥⌘S` | Пропустить текущую фазу |
| `⌃⌥⌘R` | Сброс |

Требуется разрешение Универсального доступа в Системных настройках.

## Функции

- **Выпадающая панель**: крупный 32pt обратный отсчёт, управление, статистика за день, недельный график
- **12 звуков тиканья**: с предпрослушиванием в настройках
- **Уведомления**: звук+баннер / только звук / только баннер / выкл
- **Статистика**: локальное JSON-хранилище
- **Сохранение состояния**: при каждой смене фазы, точное восстановление после перезапуска
- **11 языков**: автоопределение или ручной выбор

## Архитектура

7 файлов Swift, ноль внешних зависимостей. Рекурсивный enum `TimerPhase` для конечного автомата, `NotificationCenter` для разделения модулей, `@AppStorage` для настроек.

## FAQ

**Иконка не отображается?** macOS 26+: Системные настройки → Пункт управления → включите PomodoroNotch.

**Горячие клавиши не работают?** Проверьте разрешение Универсального доступа.
